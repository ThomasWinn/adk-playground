# adk-playground
A playground for Google's Agent Development Kit (ADK)

### Installation
python = 3.12

### Setup
ANTHROPIC_API_KEY=(in .env)

### How to Use
At top level ```/adx-playground```
```
adk web
```

Then go to 
```
http://localhost:8000
```

### Quit Program
On a separate terminal
```
ps aux | grep adk
```

Then 
``` 
kill <PID_#>
```
